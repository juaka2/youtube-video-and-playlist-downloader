# YOUTUBE VIDEO/PLAYLIST DOWNLOADER.
open main.py and enter ur url playlist or video
wait 1-2+ minutes and ur video downloaded to the folder "videos.видео"
script have 2 language (russian, english) 
to change ur language open language.json and enter this to change language to russian (idk why do you need this, because the script puts English by default)
{
    "language": "ru"
}
